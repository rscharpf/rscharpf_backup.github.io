Research Overview

I develop statistical methods and software for the analysis of high-throughput genomic data, including methods for estimation of cross-study differential gene expression from mRNA microarrays and detection of DNA structural variants from array- and sequence-based platforms. I also develop methods and software for preprocessing array and sequence platforms to remove batch effects and other technical sources of variation. Software is available from Bioconductor, with developmental versions available from github.

Bioconductor Packages

  • oligoClasses: infrastructure for crlmm and oligo packages
  • crlmm: Methods for calling genotypes and copy number for Affymetrix and Illumina genotyping platforms
  • SNPchip: A few plotting utilities for log R ratios / BAFs; chromosome idiograms
  • VanillaICE: A hidden Markov model for inferring copy number alterations from genotyping arrays
  • XDE: A Bayesian Hierarchical model for cross-study differential gene expression
  • MinimumDistance: Methods for calling de novo copy number variants in case-parent trios
  • ArrayTV: Software for removing genomic waves from marker-level estimates of copy number such as log R ratios derived Affymetrix or Illumina genotyping arrays

The above packages can be installed at the R prompt by

> source("http://www.bioconductor.org/biocLite.R")
> pkgs <- c("oligoClasses, "crlmm", "XDE", "MinimumDistance", "ArrayTV", "SNPchip", "VanillaICE")
> biocLite(pkgs)

GitHub Packages

The developmental versions of these packages (use at your own risk) are available on github can be installed by

> install.packages("devtools")
> library(devtools)
> install_github("MinimumDistance")

Note these packages are tested against the developmental version of R.
